## 📖 About the Project

This project implements a **College FAQ Chatbot** that can answer student queries like:
- What are the college timings?
- How can I apply for admission?
- What courses are offered?
- Is there a hostel facility?
- What is the fee structure?

Instead of hard-coded rules, the chatbot uses **RAG (Retrieval-Augmented Generation)** — it semantically searches a knowledge base and generates human-like answers using an open-source LLM.

---

## 🏗️ How It Works

```
User Query
    ↓
Encode query using Sentence Transformer (all-MiniLM-L6-v2)
    ↓
Search FAISS vector index for top-k similar documents
    ↓
Build prompt with retrieved context
    ↓
Generate answer using google/flan-t5-base
    ↓
Print response to user
```

## ⚙️ Experimentation

You can experiment with the following parameters:

### 1. Number of Retrieved Documents (`top_k`)
```python
generate_answer("Your question", top_k=1)  # precise
generate_answer("Your question", top_k=2)  # balanced (default)
generate_answer("Your question", top_k=3)  # more context
```

### 2. LLM Temperature
```python
# More deterministic
llm = pipeline("text-generation", model="google/flan-t5-base", temperature=0.1)

# More creative
llm = pipeline("text-generation", model="google/flan-t5-base", temperature=1.0)
```

### 3. Embedding Model
```python
# Faster, lighter (default)
embedding_model = SentenceTransformer("all-MiniLM-L6-v2")

# Slower, more accurate
embedding_model = SentenceTransformer("all-mpnet-base-v2")
```
